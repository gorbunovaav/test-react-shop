# Описание

Небольшое SPA-приложение магазина товаров на React с фильтрацией, анимациями и модальным окном.

# Реализованные функции

- Загрузка товаров с публичного API (FakeStore API)
- Фильтрация товаров по категориям
- Анимированное появление карточек (Framer Motion)
- Модальное окно с детальной информацией о товаре
- Плавное раскрытие длинного описания в модалке
- Адаптивная сетка карточек
- Loader-спиннер во время загрузки данных


## Архитектура проекта

Проект построен по компонентному принципу:

### Основные решения
- React + Hooks (`useState`, `useEffect`)
- Framer Motion — для анимаций карточек и модального окна
- Axios — для работы с API
- SCSS 
- Vite — быстрый сборщик и dev-сервер
